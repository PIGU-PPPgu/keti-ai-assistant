## 申报书结构

### 一、基本信息
- 课题名称
- 主题词（5个）
- 主持人信息
- 团队成员（最多8人）
- 预期成果
- 预计完成时间

### 二、课题设计论证（5000字以内）
1. 选题（研究现状、背景、意义）
2. 内容（研究思路、方法、内容）
3. 预期价值（理论创新、应用价值）

### 三、完成项目的可行性
- 研究基础
- 研究条件
- 团队优势

### 四、研究工作进度
- 阶段性成果（3-4个阶段）
- 最终成果

### 五、经费预算
- 经费来源：学校自筹
- 年度预算：合理分配

### 六、审核意见
- 学校审核
- 区级审核

## 课题命名模板

### 专项课题
```
[选题指南题目] + [细化方向]

示例：
选题指南: "AI赋能下传统节日文化融入小学道德与法治课程的实践研究"
生成:
- "AI赋能下端午节文化融入小学三年级道德与法治课程的实践研究"
- "基于AI技术的春节文化融入小学高年级道德与法治课程策略研究"
```

### 自选课题
```
[热点关键词] + [学科] + [教学痛点] + [研究方法]

示例：
- "核心素养导向的初中数学大单元教学实践研究"
- "双减背景下小学语文作业设计优化策略研究"
- "数字化转型中高中英语混合式教学模式研究"
```
