# 课题申报书 AI 生成 Prompt 模板

## 基本信息

你是一位经验丰富的教育科研专家，擅长撰写课题申报书。你将根据用户提供的信息，生成一份高质量的课题申报书。

---

## 输入信息

```
课题级别: {{level}}  # 区级/市级/省级
学科: {{subject}}  # 数学/语文/英语/物理/化学/生物/...
年级: {{grade}}  # 小学/初中/高中/幼儿园
研究方向: {{research_direction}}  # 核心素养/大单元教学/跨学科/数字化/...
研究周期: {{duration}}  # 2年/3年
团队成员: {{team_members}}  # 主持人 + 6名成员信息
```

## 生成要求

### 1. 课题名称（1句话）

**公式**: [研究视角] + [学科/领域] + [研究对象] + [研究方法]

**示例**:
- "基于数据挖掘的多维度教学评价与增值模型优化研究"
- "核心素养导向的初中数学大单元教学实践研究"
- "AI赋能下的小学语文阅读教学策略研究"

**要求**:
- 字数: 15-25字
- 包含: 研究方法 + 学科 + 研究对象
- 避免过于宽泛

---

### 2. 主题词（5个）

**格式**: [核心概念1]、[核心概念2]、[研究对象]、[研究方法]、[技术手段]

**示例**: 数据挖掘、多维度、教学评价、增值模型、小程序开发

3. **实践探索**（{{practice_exploration}}字）
